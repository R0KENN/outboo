# Telegram-бот для модерации и автопостинга

Модульный бот-движок на **aiogram 3.x** для автоматической модерации чатов,
управления новичками, отложенного автопостинга в каналы и сбора статистики.
Реализован по ТЗ v1.0 (Этапы 1–4).

---

## Возможности

**Модерация (раздел 4.1 ТЗ)**
Антиспам (ссылки, форварды, белый список доменов), антимат с обходом замен
(leet, разделители), антифлуд (N сообщений за M секунд с автомутом), система
предупреждений с автодействием при достижении порога, ручные команды.

**Новички (раздел 4.2 ТЗ)**
Капча (кнопка или математический пример) с киком по таймауту, приветствие с
подстановкой `{name}` и автоудалением, карантин новых аккаунтов, очистка
служебных сообщений о входе/выходе.

**Автопостинг (раздел 4.3 ТЗ)**
Создание отложенного поста через диалог в личке, медиа (фото/видео/документ/
аудио/альбом), HTML-форматирование, inline-кнопки, публикация в несколько
каналов, таймер автоудаления, управление очередью, повторы (daily/weekly).

**Администрирование (раздел 4.4 ТЗ)**
Панель настроек на inline-кнопках (per-chat), роли младших модераторов с
ограниченным набором прав, журнал всех модераторских действий.

**Статистика (раздел 4.5 ТЗ)**
Счётчики по чату (новые участники, удаления по фильтрам, баны/муты/варны,
активность) с выводом за день/неделю/месяц.

## Технологический стек

| Компонент       | Назначение                                  |
|-----------------|---------------------------------------------|
| Python 3.11+    | Язык                                        |
| aiogram 3.x     | Асинхронный фреймворк Telegram Bot API      |
| SQLAlchemy 2.x  | ORM (переносимость между БД)                 |
| SQLite          | БД по умолчанию (одиночная установка)       |
| PostgreSQL      | БД для высокой нагрузки (опционально)       |
| APScheduler 3.x | Планировщик отложенных постов               |
| Alembic         | Миграции БД                                 |
